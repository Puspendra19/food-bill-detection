# food-bill-detection
A Python-based OCR project that detects food items and prices from restaurant bills using Tesseract OCR and displays the results through a Streamlit UI.
## Features
- Upload restaurant bill image
- Extract text using OCR
- Detect food items and their prices
- Ignore payment details like UPI, VISA, etc.
- Display extracted items in a table
- Download detected bill items as CSV

## Technologies Used
- Python
- Tesseract OCR
- Streamlit
- Pandas
- Regex (Regular Expressions)
- Pillow (Image processing)
